# CareerForge AI

CareerForge AI is a local-first interview preparation workspace. It now supports pasted resume text, multi-format resume file upload, bilingual deep resume analysis, mock interviews, knowledge-base Q&A, and interview schedule parsing.

## What Changed In This Iteration

- Added two resume input modes: pasted text and file upload.
- Added upload parsing for `.pdf`, `.doc`, `.docx`, `.txt`, `.zip`, and `.rar`.
- Added recursive archive parsing with automatic filtering for hidden files, images, and unsupported files.
- Added bilingual resume analysis modes: auto language detection, forced Chinese, and forced English.
- Replaced the old simple scoring logic with a deep 7-section resume review model.
- Added OpenAI-compatible AI API configuration with local deep-analysis fallback.
- Added file-size and request-size limits with friendly Chinese error messages.
- Added Chinese-first UI, UI language setting, light/dark/system theme modes, and a light-blue visual refresh.
- Added clipboard upload for resume files or pasted resume text.
- Expanded the knowledge base into a local interview evidence library with categories, citations, confidence, and follow-up suggestions.
- Split the knowledge base into `我的资料` and `公开资料库`.
- Added a generated public interview-material library covering interview Q&A, job descriptions, company research, project retrospectives, resume material, and notes.
- Added category-aware suggested questions that can be clicked to ask immediately.
- Upgraded knowledge-base answers to DeepSeek Flash and DeepSeek Pro.
- Added question history cards with horizontal navigation, copy answer, and follow-up actions.
- Added DeepSeek-powered mock interview question generation, answer evaluation, and interview summaries with local fallback.
- Added monthly usage tracking for each DeepSeek model: 1000 free answers per model for normal users, placeholder overage pricing, and unlimited admin usage.

## Resume Analysis Sections

1. Basic information completeness
2. Work/internship experience review
3. Project experience depth review
4. Skill stack match review

## Install

```bash
cd careerforge-ai
npm install
```

New parsing dependencies:

- `pdf-parse`: PDF text extraction
- `mammoth`: DOCX text extraction
- `word-extractor`: legacy DOC text extraction
- `jszip`: ZIP archive parsing
- `node-unrar-js`: RAR archive parsing

## Run Locally

```bash
npm run dev
```

Open http://localhost:4173.

## UI Language And Theme

Open `设置` and choose:

- `界面语言`: 中文, English, 日本語, 한국어, Français, Español. Chinese and English have full UI labels; other options are reserved for common-language expansion while resume parsing still accepts multilingual text content.
- `主题模式`: 跟随系统, 浅色模式, 深色模式.

The default UI is Chinese and the visual style uses a light-blue theme.

## Resume Input Modes

Open `简历分析` and choose one of three input modes:

- `粘贴文本`: paste the resume body directly.
- `上传文件`: choose PDF, DOC, DOCX, TXT, ZIP, or RAR files.
- `粘贴上传`: copy files from the file manager and press Ctrl+V in the paste area, or paste raw resume text. Pasted text is analyzed as a virtual TXT resume.

## Knowledge Base

The knowledge base is split into two parts:

- `我的资料`: private material imported by the user.
- `公开资料库`: built-in general interview preparation material. It is generated from original templates and common interview-preparation patterns, not copied from third-party copyrighted pages.

You can store or import:

- interview questions and model answers
- target job descriptions
- company and business notes
- project retrospectives
- resume material
- general notes

When you ask a question, CareerForge retrieves matching snippets from `我的资料`, `公开资料库`, or both, then generates a complete answer with one of two DeepSeek models:

- `DeepSeek Flash`: faster answers for daily review and quick interview preparation.
- `DeepSeek Pro`: deeper reasoning for project drilling, difficult interview questions, and stronger answer review.

The answer UI stores recent questions as horizontal cards. Each card can be copied or used as the basis for a follow-up question. Model output is cleaned into normal readable paragraphs instead of raw Markdown symbols.

If a DeepSeek/OpenAI-compatible API is configured, the knowledge-base answer is generated by the selected model. If no API key is configured or the model call fails, the app falls back to a local complete-answer generator so the answer area remains usable.

Each model has a monthly free quota of 1000 answers for normal users. Placeholder overage pricing is currently:

- DeepSeek Flash: 0.05 CNY per extra answer
- DeepSeek Pro: 0.15 CNY per extra answer

Admins have unlimited usage and are not counted. If an external AI provider is connected, the actual model cost still follows that provider's billing rules; CareerForge only tracks answer counts and displays local placeholder overage estimates.

## Mock Interview AI

`模拟面试` also supports DeepSeek Flash and DeepSeek Pro. The selected model is used for:

- generating interview questions from the role and job description
- evaluating submitted answers
- producing the final interview summary

If the model is unavailable, the system automatically falls back to the local interview question bank and local scoring rules.

## Configure AI

The project runs without an API key by using the built-in local deep analyzer and local complete-answer fallback. To enable DeepSeek-powered resume analysis, knowledge-base answers, and mock interviews, copy `.env.example` to `.env` and fill:

```env
AI_PROVIDER=openai-compatible
AI_API_BASE_URL=https://api.deepseek.com/chat/completions
AI_API_KEY=your_deepseek_api_key_here
AI_MODEL_FLASH=deepseek-v4-flash
AI_MODEL_PRO=deepseek-v4-pro
AI_DEFAULT_MODEL=deepseek-v4-flash
AI_TIMEOUT_MS=60000
```

Any OpenAI-compatible chat-completions endpoint can still be used by changing the base URL and model names.

## Upload Limits

```env
MAX_UPLOAD_MB=20
MAX_REQUEST_MB=35
```

`MAX_UPLOAD_MB` applies to each uploaded file. `MAX_REQUEST_MB` applies to the JSON request body after browser base64 encoding.
